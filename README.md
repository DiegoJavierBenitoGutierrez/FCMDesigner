# FCM Designer

A graphical tool for the design, simulation, and learning of Fuzzy 
Cognitive Maps (FCMs) and Dynamic Fuzzy Cognitive Maps (DFCMs).

## Download

- [FCMDesigner.exe](dist/FCMDesigner.exe) — Windows executable
- [User Guide](docs/user_guide.pdf) — Installation and usage instructions

## Getting Started

Download the executable and refer to the user guide for installation 
and usage instructions. No additional dependencies are required.

## Features

- Interactive graphical editor for FCM and DFCM construction
- Support for multiple inference rules and activation functions
- Data-driven weight learning via Particle Swarm Optimization (PSO)
- Binary and multiclass classification support
- Export of results in CSV and Excel formats
